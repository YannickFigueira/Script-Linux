###Configurações iniciais do servidor###
Configurar o openssh na instalação do Ubuntu Server
Acessar pelo ssh para facilitar o download dos arquivos e depois executar de dentro do sistema

Baixar arquivos de configuração

### Acessar servidor via ssh link de downlod do arquivo de configuração automatizada
wget -O raid_config.sh https://raw.githubusercontent.com/YannickFigueira/Script-Linux/main/Ubuntu%20Server/configuration/raid_config.sh
wget -O server_config.sh https://raw.githubusercontent.com/YannickFigueira/Script-Linux/main/Ubuntu%20Server/configuration/server_config.sh

## inicio da configuração para um servidor novo
# Liberar arquivo para ser usado como script
chmod +x raid_config.sh
chmod +x server_config.sh

# Executar arquivos em ordem
./raid_config.sh
./server_config.sh

# Após a execução
# bash
source /home/$USER/.bashrc
